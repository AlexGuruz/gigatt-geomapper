Python not found

GIGATT Geomapper needs Python 3 to run. Install it or use a portable copy.

Option 1: Install Python (recommended)

  • Download the Windows installer from python.org/downloads.
  • Run the installer and check “Add Python to PATH”.
  • Then run Start GIGATT Geomapper.bat again.

Option 2: Portable Python (no install)

  • Download the Windows embeddable package (e.g. “Python 3.12.x - Windows embeddable package (64-bit)”).
  • Extract it into a folder named python inside this app folder (same folder as the .bat file).
  • You should have python\python.exe in this directory.
  • Then run Start GIGATT Geomapper.bat again.
